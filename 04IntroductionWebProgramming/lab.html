2.Differentiation

Tag เหมือนกัน แตกต่าง
hr & br เป็นการแบ่งเนื้อหาข้อมูลออกเหมือนกัน hr = แบ่งโดยใช้เส้นขีดยาว , br = แบ่งโดยการเว้นบรรทัด
ul & ol เป็นการเรียงข้อมูลออกมาในรูปแบบลิสเหมือนกัน ul คือ unordered list มีสัญลักษณ์ต่างๆที่ไม่ใช้เลข
ol คือ ordered list จะเรียงโดยใช้ตัวเลข
select & checkbox & radio button เป็นformตัวเลือกเหมือนกัน select เป็นรูปแบบของ dropdown lists มีตัวเลือกลงมา
checkbox เป็นรูปแบบของการกดเลือกในกล่องสี่เหลี่ยมซึ่งสามารถเลือกได้มากกว่าหนึ่งช่อง
radio button เป็นรูปแบบของการกดเลือกในกล่องสี่เหลี่ยมได้เพียงกล่องเดียว
div & span เป็น Element เหมือนกัน div เป็น block element จะถูกจัดเรียงกันลงมาคนละบรรทัด
span เป็น inline element จะถูกจัดเรียงติดกันโดยอยุ่ในบรรทัดเดียวกัน
script & style เป็น internal tag ที่ใช้ในไฟล์ html เหมือนกัน script เป็นแทคสำหรับการเขียน javascript
style เป็นแทคสำหรับการตกแต่งกับ css



3.Input type

Input Type การใช้งาน
input type="text" กำหนดฟิลด์ป้อนข้อความบรรทัดเดียว
input type="password" กำหนดฟิลด์รหัสผ่าน
input type="submit" กำหนดปุ่มสำหรับส่งข้อมูลแบบฟอร์มไปยัง แบบฟอร์มการจัดการ
แบบฟอร์มการจัดการ เป็นหน้าเซิร์ฟเวอร์ที่มีสคริปต์สำหรับประมวลผลข้อมูลอินพุต
แบบฟอร์มการจัดการ ถูกระบุในแอ็ตทริบิวต์ action ของฟอร์ม
input type="reset" กำหนดปุ่มรีเซ็ตที่จะรีเซ็ตค่าแบบฟอร์มทั้งหมดเป็นค่าเริ่มต้น
input type="radio" กำหนดปุ่มตัวเลือก
ปุ่มตัวเลือกให้ผู้ใช้เลือกหนึ่งตัวเลือกในจำนวนจำกัดเท่านั้น
input type="checkbox" กำหนดช่องทำเครื่องหมาย
ช่องทำเครื่องหมายให้ผู้ใช้เลือก ไม่เลือกเลย หรือ เลือกเพิ่มเติมของตัวเลือกทั้งหมด
input type="button" กำหนดปุ่มกด
input type="color" ใช้สำหรับฟิลด์อินพุตที่ควรมีสี
ตัวเลือกสีสามารถแสดงขึ้นในช่องป้อนข้อมูลทั้งนี้ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์
input type="date" ใช้สำหรับฟิลด์อินพุตที่ควรมีวันที่
ตัวเลือกวันที่สามารถแสดงขึ้นในฟิลด์ป้อนข้อมูลทั้งนี้ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์
input type="datetime-local" ระบุเขตข้อมูลวันที่และเวลาโดยไม่มีเขตเวลา
ตัวเลือกวันที่สามารถแสดงขึ้นในฟิลด์ป้อนข้อมูลทั้งนี้ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์
input type="email" ใช้สำหรับฟิลด์อินพุตที่ควรมีที่อยู่อีเมล
ที่อยู่อีเมลสามารถตรวจสอบได้โดยอัตโนมัติเมื่อส่ง
input type="file" กำหนดฟิลด์เลือกไฟล์และปุ่ม "เรียกดู" สำหรับการอัพโหลดไฟล์
input type="month" อนุญาตให้ผู้ใช้เลือกเดือนและปี
ตัวเลือกวันที่สามารถแสดงขึ้นในฟิลด์ป้อนข้อมูลทั้งนี้ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์
input type="number" กำหนดฟิลด์อินพุตเป็นตัวเลข
นอกจากนี้คุณยังสามารถกำหนดข้อ จำกัด เกี่ยวกับหมายเลขที่ยอมรับได้
input type="range" กำหนดตัวควบคุมสำหรับการป้อนตัวเลขซึ่งค่าที่ไม่สำคัญ
(เช่นตัวควบคุมแถบเลื่อน) ช่วงเริ่มต้นคือ 0 ถึง 100 อย่างไรก็ตามคุณสามารถกำหนดข้อ จำกัด เกี่ยวกับตัวเลขที่ยอมรับได้ด้วยแอตทริบิวต์ min, max และ step
input type="search" ใช้สำหรับช่องค้นหา (ช่องค้นหาจะทำงานเหมือนกับช่องข้อความทั่วไป)
input type="tel" ใช้สำหรับฟิลด์อินพุตที่ควรมีหมายเลขโทรศัพท์
input type="time" อนุญาตให้ผู้ใช้เลือกเวลา (ไม่มีเขตเวลา)
ตัวเลือกเวลาสามารถปรากฏขึ้นในฟิลด์อินพุตทั้งนี้ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์
input type="url" ใช้สำหรับฟิลด์อินพุตที่ควรมีที่อยู่ URL
ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์ฟิลด์ url สามารถตรวจสอบความถูกต้องโดยอัตโนมัติเมื่อส่ง
input type="week" อนุญาตให้ผู้ใช้เลือกหนึ่งสัปดาห์และหนึ่งปี
ตัวเลือกวันที่สามารถแสดงขึ้นในฟิลด์ป้อนข้อมูลทั้งนี้ขึ้นอยู่กับการสนับสนุนของเบราว์เซอร์
